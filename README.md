# Sobre o projeto

Este é um projeto de Landing Page Data Science.

O projeto foi estruturado, a partir da atividade proposta do módulo ReactJS, Frontend 3, da formação STARTER FULL STACK WEB DEVELOPER, pela @Growdev.
